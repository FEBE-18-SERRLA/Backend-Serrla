# API Specification

## Authentication

All API must use this authentication

Request :

- Header :
  - X-Api-Key : "your secret api key"

## Login

Request :

- Method : POST
- Endpoint : `api/auth/login`
- Header :
  - Content-Type: application/json
  - Accept: application/json
- Body :

```json
{
  "email": "string, unique",
  "password": "string"
}
```

Response :

```json
{
  "code": "number",
  "status": "string",
  "token": "number"
}
```
